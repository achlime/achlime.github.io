Compendium Infernal

Any Shintai Charm

Source: The Broken-Winged Crane (p. 44)

This prerequisite is satisfied if the Infernal knows any of the alternative Charms listed. Note that all alternative shintai listed here belong to a Yozi—only those Shintai can be used to satisfy this prerequisite, not any Charm that happens to be named “Shintai”.

Possible alternatives:
  • Devil-Tyrant Avatar Shintai
  • Demon Emperor Shintai
  • Soul-Sand Devil Shintai
  • Greater Shintai of the Endless Desert
  • Heuristic Logos Shintai
  • Splintered Gale Shintai
  • Scarlet Rapture Shintai
  • Black Mirror Shintai
  • All-Devouring Depths Shintai
